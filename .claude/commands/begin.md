# Purpose

Begin the workshop on a clean `module-1` branch, leaving any setup or configuration
changes behind so module-1 can be focused entirely on the workshop content.

Unlike `/module`, this command does **not** merge the current branch into `module-1`.

## Instructions

1. Confirm with the user that they are ready to begin the workshop.
2. Switch to `module-1`.
3. Inform the user they are ready to start Module 1.
